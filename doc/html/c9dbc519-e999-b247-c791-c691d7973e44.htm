Obtient le code status du lecteur.

 

Syntaxe

C#
public int status { get; set; }

Valeur de propriété

Le code status du lecteur.

Remarques

Les codes status sont les suivants :
0OK ( Ok ).
1ServiceKO ( Problème du service utilisant le lecteur ). Exemple agent de communication.
2ReadMute ( Problème lecture carte ), lorsque le lecteur ne lit plus les cartes.
3InsertKO ( Problème insertion carte ), lorsque le lecteur ne voit plus l'insertion / retrait cartes.
