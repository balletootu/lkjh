ipm..::..tabs Class
ipm Class Membres Exemples Voir aussi 
Fournit les propriétés et méthodes de l'api relative aux tabs du navigateur.

 

Exemples

JavaScript Copy
var ipm = window.external.ipm;
var tab = ipm.tabs.get('tabname');  // le tab spécifié par son nom
var tab = ipm.tabs.me;  // le tab courant
var browser = ipm.tabs.browser;  // le tab 'browser', tab principal de navigation
var oskbd = ipm.tabs.oskbd;  // le tab 'oskbd', clavier visuel
var splash = ipm.tabs.splash;  // le tab 'splash', affiché au démarrage
