Obtient le code erreur de l'imprimante.

 

Syntaxe

C#
public int error { get; }

Valeur de propriété

Le code erreur de l'imprimante.

Remarques

Les codes erreur sont les suivants :
0Ok ( Pas d'erreur ).
1Offline ( Hors connexion ).
2NearPaperEnd ( Fin de papier proche ). Simple avertissement, détecté par une cellule.
3PaperEnd ( Fin de papier ). Plus de papier au niveau de la tête d'impression de l'imprimante.
4PaperJam ( Bourrage papier ).
5DoorOpen ( Porte ouverte ).
6Other ( Autre erreur ).
Lorsqu'une erreur survient, l'événement ipmOnPrinterErrorChanged est généré.
Note
L'imprimante est en état de marche avec les valeurs Ok ou NearPaperEnd.
