Obtient ou définit le statut de l'application.

 

Syntaxe

C#
public int appStatus { get; set; }

Valeur de propriété

Remarques

Le status prend les valeurs suivantes :
-1Application non initialisée.
0Application initialisée et opérationnelle.
xAutres valeurs positives, à la discrétion de l'application.

Exemples

JavaScript Copy
var ipm = window.external.ipm;
var browser = ipm.tabs.browser;  // le tab 'browser'
browser.appStatus = 0;  // pour indiquer la fin de l'intialisation de l'application
