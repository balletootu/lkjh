Méthodes

  Name Description
cancelAllJobs
Vide le spooler de l'imprimante.
printFrame
Not Implemented. Lance l'impression de la frame spécifiée.
printFromCacheLastDocument
Lance l'impression du dernier document spécifié du cache.
printPage
Lance l'impression de la page spécifiée.
printPdfInFrame
Not Implemented.
printScreen
Not implemented. Lance une copie d'écran sur l'imprimante.
printTab
Lance l'impression du contenu du tab spécifié.
printTestPage
Lance l'impression de la page de test de l'imprimante (debug).
refreshStatus
Force un rafraîchissement de l'état de l'imprimante.
reset
Réservé imprimante Custom : lance un reset de l'imprimante.
setDefault
Positionne l'imprimante comme imprimante par défaut pour le système.

Propriétés

  Name Description
customDllLastError
Obtient le code retour du dernier appel au driver d'une imprimante Custom.
customStatus
Obtient le code status retourné par le driver d'une imprimante Custom.
customStatusList
Obtient la liste des status positionnés par le driver d'une imprimante Custom.
error
Obtient le code erreur de l'imprimante.
firmwareVersion
Obtient la version du firmware de l'imprimante.
isDefault
Indique si l'imprimante est définie comme imprimante par défaut par le sytème.
jobId
Obtient l'id du job d'impression en cours (0 si pas d'impression).
jobPagesPrinted
Obtient le nombre de pages envoyées à l'imprimante.
jobs
Obtient le nombre de jobs dans le spooler d'impression.
jobTotalPages
Obtient le nombre de pages restant à envoyer à l'imprimante pour le document en cours.
name
Obtient le nom système de l'imprimante.
paperReams
Réservé imprimante Custom : Le nombre total de changements de ramettes de papier.
printerReplace
Réservé imprimante Custom : Le nombre total de changements d'imprimantes.
printingError
Obtient le code erreur d'impression éventuelle (debug).
state
Obtient le code status wmi de l'imprimante.
status
Obtient le code status d'impression de l'imprimante.
strCustomStatus
Obtient la propriété customStatus de l'imprimante sous forme littérale hexadécimale.
strError
Obtient la propriété error de l'imprimante sous forme littérale.
strPrintingError
Obtient la propriété printingError de l'imprimante sous forme littérale.
strState
Obtient la propriété state de l'imprimante sous forme littérale hexadécimale.
strStatus
Obtient la propriété status de l'imprimante sous forme littérale.
todayCutPages
Réservé imprimante Custom : lit le nombre de pages coupées dans la journée.
todayRetractedPages
Réservé imprimante Custom : lit le nombre de pages rétractées dans la journée.
totalCutPages
Réservé imprimante Custom : lit le nombre total de pages coupées.
totalJobsPrinted
Obtient le nombre de documents imprimés depuis le démarrage de la borne.
totalPagesPrinted
Obtient le nombre de pages imprimées depuis le démarrage de la borne.
totalRetractedPages
Réservé imprimante Custom : lit le nombre total de pages rétractées.
trashEmpty
Réservé imprimante Custom : Le nombre total de vidages de la corbeille de pages rétractées.

ipmOnPrinterJobsChanged
Se produit lorsque la valeur de la propriété jobs est modifié.
ipmOnPrinterPrintingError
Se produit lorsque la propiété printingError indique une erreur.
ipmOnPrinterStatusChanged
Se produit lorsque la valeur de la propriété status de l'imprimante est modifiée.
