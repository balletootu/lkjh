Obtient l'imprimante printer à partir de son nom système.

 

Syntaxe

C#
public printer get(
	string name = null
)

Paramètres

name (Optional)
Nom système Windows de l'imprimante.

Valeur de retour

L'imprimante printer ou null si l'imprimante n'existe pas.

Exemples

Pour obtenir l'imprimante par défaut.
JavaScript Copy
var ipm = window.external.ipm;
var printer = ipm.printers.get();
if (printer) { ... }
Pour obtenir l'imprimante CUSTOM TG2480-H.
JavaScript Copy
var ipm = window.external.ipm;
var printer = ipm.printers.get("CUSTOM TG2480-H");
if (printer) { ... }
