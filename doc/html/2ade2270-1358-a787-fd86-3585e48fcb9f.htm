Obtient le lecteur [!:PcscReader] à partir de son nom système.

 

Syntaxe

C#
public pcscReader getReader(
	string name = null
)

Paramètres

name (Optional)
Nom système Windows du lecteur.

Valeur de retour

Le lecteur [!:PcscReader] ou null si le lecteur n'existe pas.

Exemples

Pas de nom précisé, renvoie le 1er (et seul) lecteur connecté.
JavaScript Copy
var reader = window.external.ipm.pcscReaders.getReader("");
