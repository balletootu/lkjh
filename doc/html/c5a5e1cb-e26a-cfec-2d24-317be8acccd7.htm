Lance l'impression du dernier document spécifié du cache.

 

Syntaxe

C#
public bool printFromCacheLastDocument(
	string pattern
)

Paramètres

pattern
Pattern du nom de fichier.

Valeur de retour

true si l'impression est lancée, false sinon.

Remarques

Comme pour printDocument(String), cette fonction peut être complètement silencieuse (pas d'affichage de popup).

Remarques

Il n'est pas possible d'imprimer plus d'un document à la fois.

Exemples

JavaScript Copy
var printer = window.external.ipm.printers.getPrinter("");
printer.printFromCacheLastDocument('*.pdf');
