Se produit lorsque la valeur de la propriété jobs est modifié.

 

Syntaxe

C#
public static event CallbackEventHandler ipmOnPrinterJobsChanged

Prototype de la callback

void ipmOnPrinterJobsChanged ( string name )

Paramètres

name
Le nom système de l'imprimante.

Remarques

Le nombre de jobs est accessible par jobs. S'il est non nul, un nouveau document est en cours d'impression. S'il est nul, le document n'est plus en cours d'impression.
