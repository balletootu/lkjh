Obtient la liste des erreurs présentes sur la borne.

 

Syntaxe

C#
public string errors { get; }

Valeur de propriété

La liste des erreurs séparées par des espaces.

Remarques

Les erreurs peuvent être parmi les suivantes :
PaperPrinterImprimante papier.
ReceiptPrinterImprimante ticket.
TicketPrinterImprimante billet.
BarcodeReaderLecteur de codes barres.
PcscReaderLecteur pcsc.
CreditCardReaderLecteur bancaire.
BatteryOnduleur.
KeyboardClavier physique.
TouchscreenDalle tactile.
ScannerScanner de document.
WebcamCamera video.
EthernetConnexion Ethernet.
WifiConnexion wifi.
InternetLiaison internet.

Les erreurs sont renvoyées dans cet ordre.

Note
Pour une imprimante, une presque fin de papier ne remonte pas une erreur au niveau borne.

Lorsque la liste des erreurs change, l'événement [!:ipmKioskEvent.ipmOnKioskErrorsChanged] est généré.

Exemples

Pour obtenir le tableau des erreurs présentes sur la borne :
JavaScript Copy
var errors = window.external.ipm.kiosk.errors.split(' ');
