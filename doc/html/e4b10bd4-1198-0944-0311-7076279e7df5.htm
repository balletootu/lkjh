Obtient le code status d'impression de l'imprimante.

 

Syntaxe

C#
public int status { get; }

Valeur de propriété

Le code status d'impression de l'imprimante.

Remarques

Les codes status sont les suivants :
0Ready ( Prête ).
1Unknown ( Inconnu ), lorsque l'imprimante est Offline, voir error.
2Spooling ( Spooling en cours ), lorsqu'un job est dans le spooler.
3Printing ( Impression en cours ), lorsqu'un document / page est en cours d'impression.
4BinFull ( Présentoir ), lorsqu'une page est disponible.
5Deleting ( Suppression en cours ), lorsqu'un job est en cours de suppression.
6Error ( Erreur ), lorsque l'imprimante est en défaut, voir error.
Lorsque le status d'impression change, l'événement ipmOnPrinterStatusChanged est généré.

Lorsque l'imprimante gère l'impression page à page, le status passe de Printing à BinFull à chaque page imprimée, pour finir à Ready.
Sinon, le status ne progresse qu'une seule fois de Printing à Ready automatiquement en fin d'impression.

Lorsque l'imprimante gère l'impression page à page, le status reste à l'état BinFull tant que la page n'est pas retirée par l'utilisateur.
Au bout d'une temporisation donnée (typiquement 20s), la page est automatiquement rétractée et l'impression en cours annulée (status Deleting s'il restait des pages à imprimer).
