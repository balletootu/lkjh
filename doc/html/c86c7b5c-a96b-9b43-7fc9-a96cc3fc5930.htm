ipm..::..payment..::..debit Method

Lance la procédure de paiement par carte bancaire, pour le montant demandé.

 

Syntaxe

C#
public int debit(
	int amount
)

Paramètres

amount
Montant (entier positif), en centimes.

Valeur de retour

0 si le paiement a été autorisé et effectué, > 0 en cas d'erreur. Liste des codes d'erreurs, pour le lecteur WYNID :
1Paiement non effectué, impossible d'ouvrir la session monétique locale.
2Paiement abandonné par le bouton clavier.
3Paiement non autorisé.
4Paiement autorisé mais non effectué (problème durant la phase 2e phase d'enregistrement).
5Paiement effectué, mais ticket non généré (impression impossible), pb d'installation.
6Paiement abandonné par le bouton à l'écran.
7Paiement abandonné, délai dépassé.
8Montant incorrect.
100Echec de la connexion au serveur monétique distant.
101Lecteur en panne.
102Problème d'installation (fichiers de configuration).
103Autre erreur.
104Erreur dans le module lecteur.
255Erreur inattendue dans l'application paiement.

Remarques

Note

Note
Cette fonction lance la procédure de paiement de façon synchrone.
Cette procédure possède sa propre IHM qui vient en superposition du navigateur.
La procédure consiste en la demande d'autorisation puis automatiquement la demande d'enregistrement (débit effectif).
Il n'y a pas moyen d'effectuer une simple demande d'autorisation actuellement.

Note
Cette procédure n'est disponible actuellement que pour les lecteurs Wynid. Dans le fichier de configuration du lecteur PaiementCB.ini, le nomPeripherique = WYBORNE.
