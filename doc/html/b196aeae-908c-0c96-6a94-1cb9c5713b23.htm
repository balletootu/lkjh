Remarques

Configuration du ticket

Le ticket imprimé est mis en forme en html et peut être personnalisé dans le ficheir de configuration ticket.html.
Le mot clé #_TICKET_# est remplacé par le contenu du ticket généré par l'exécutable de paiement.
Il faut placer ce mot clé en tout début de ligne, sinon les espaces indenteraient d'autant les informations imprimées.

Il est ainsi possible de rajouter une image (fichier local sur la borne) ou tout texte supplémentaire.
