Obtient le code status wmi de l'imprimante.

 

Syntaxe

C#
public int state { get; }

Valeur de propriété

Le code status wmi de l'imprimante (à ne pas confondre avec le status d'impression).

Remarques

Le code status wmi est une combinaison de bits, cf (msdn) printer status constants :
0x00000001 = PRINTER_STATUS_PAUSEDThe printer is paused.
0x00000002 = PRINTER_STATUS_ERRORThe printer is in an error state.
0x00000004 = PRINTER_STATUS_PENDING_DELETIONThe printer is being deleted as a result of a client's call to RpcDeletePrinter. No new jobs can be submitted on existing printer objects for that printer.
0x00000008 = PRINTER_STATUS_PAPER_JAMPaper is stuck in the printer.
0x00000010 = PRINTER_STATUS_PAPER_OUTThe printer is out of paper.
0x00000020 = PRINTER_STATUS_MANUAL_FEEDThe printer is in a manual feed state.
0x00000040 = PRINTER_STATUS_PAPER_PROBLEMThe printer has an unspecified paper problem.
0x00000080 = PRINTER_STATUS_OFFLINEThe printer is offline.
0x00000100 = PRINTER_STATUS_IO_ACTIVEThe printer is in an active input or output state.
0x00000200 = PRINTER_STATUS_BUSYThe printer is busy.
0x00000400 = PRINTER_STATUS_PRINTINGThe printer is printing.
0x00000800 = PRINTER_STATUS_OUTPUT_BIN_FULLThe printer's output bin is full.
0x00001000 = PRINTER_STATUS_NOT_AVAILABLEThe printer is not available for printing.
0x00002000 = PRINTER_STATUS_WAITINGThe printer is waiting.
0x00004000 = PRINTER_STATUS_PROCESSINGThe printer is processing a print job.
0x00008000 = PRINTER_STATUS_INITIALIZINGThe printer is initializing.
0x00010000 = PRINTER_STATUS_WARMING_UPThe printer is warming up.
0x00020000 = PRINTER_STATUS_PAPER_OR_TONER_LOWThe printer is low on toner or on paper.
0x00040000 = PRINTER_STATUS_NO_TONERThe printer is out of toner.
0x00080000 = PRINTER_STATUS_PAGE_PUNTThe printer cannot print the current page.
0x00100000 = PRINTER_STATUS_USER_INTERVENTIONThe printer has an error that requires the user to do something.
0x00200000 = PRINTER_STATUS_OUT_OF_MEMORYThe printer has run out of memory.
0x00400000 = PRINTER_STATUS_DOOR_OPENThe printer door is open.
0x00800000 = PRINTER_STATUS_SERVER_UNKNOWNThe printer status is unknown.
0x01000000 = PRINTER_STATUS_POWER_SAVEThe printer is in power-save mode.
0x02000000 = PRINTER_STATUS_SERVER_OFFLINEThe printer is offline.
